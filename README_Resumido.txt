***Instruções para compilar e executar o código:

1°: Crie um projeto do tipo glut no CodeBlocks
2°: Copie o código do arquivo "main.c" do projeto baixado
3°: Apague todo o código da main do projeto criado
4°: Cole o código copiado na main do projeto criado
5°: Crie um arquivo denominado "carregartextura.h" no projeto criado
6°: Copie o código do arquivo "carregartextura.h" do projeto baixado
7°: Cole o código copiado no passo anterior no arquivo "carregartextura.h" do projeto criado
8°: Crie um arquivo denominado "carregartextura.cpp" no projeto criado
9°: Copie o código do arquivo "carregartextura.cpp" do projeto baixado
10°: Cole o código copiado no passo anterior no arquivo "carregartextura.cpp" do projeto criado
11°: Coloque as imagens da pasta "Texturas" do projeto baixado na pasta onde fica o executável do projeto criado no seu sistema operacional (Normalmente em PastaDoProjeto/Bin/Debug)
12°: Compile e execute o programa
13°: Feche a execução
14°: Abra o executável "TP2.exe" criado na pasta do projeto criado (Normalmente em PastaDoProjeto/Bin/Debug)
15°: Agora só se divertir



***Instruções para usar na animação:

[a] -> Ativa ou desativa o plano orbital
[s] -> Liga ou desliga a luz do Sol (Caso a iluminação estiver desligada)
[o] -> Mostrar ou esconder as órbitas dos planetas
[I] -> Mudar o tipo de iluminação (Há três) (Caso a iluminação estiver ligada)
[i] -> Ligar ou desligar a iluminação
[z] -> Mostrar ou esconder os nomes dos planetas e satélites naturais
[p] -> Pausar ou despausar o movimento dos planetas (a animação)
[F1] -> Movimentar o "olhar da câmera" no eixo Y
[F2] -> Movimentar o "olhar da câmera" no eixo Y
[PAGE UP] -> Movimentar o "olhar da câmera" no eixo X
[PAGE DOWN] -> Movimentar o "olhar da câmera" no eixo X
[HOME] -> Movimentar o "olhar da câmera" no eixo Z
[END] -> Movimentar o "olhar da câmera" no eixo Z
[UP] (Seta para cima) -> A câmera é deslocada para câmera 1
[LEFT] (Seta para esquerda) -> A câmera é deslocada para câmera 2
[DOWN] (Seta para baixo) -> A câmera é deslocada para câmera 3

[BOTÃO ESQUERDO MOUSE] -> Zoom In
[BOTÃO DIREITO MOUSE] -> Zoom Out

***Lista de extras:

1) Há skyboxes
2) Há um modo de órbitas visíveis, em que pode ser exibido ou escondido caso pressione "o"
3) Há satélites naturais: na Terra (Lua), em Marte (Phobos) e em Saturno (Titã)
4) Há um cinturão de asteroides
5) Há anéis planetários em Júpiter, Saturno, Urano e Netuno
6) Há atmosferas em Vênus e na Terra (Esferas com o raio um pouco maior e um pouco transparente com a atmosfera como textura)
7) Tem "reshape"
8) A câmera pode se deslocar para até três posições de câmeras diferentes (1, 2 e 3)
9) Há um efeito visual de calor sendo emitido pelo Sol (Uma esfera um pouco maior com um pouco de transparência)
10) Pode exibir ou esconder o plano orbital, pressionando a letra "a"
11) Podemos ver os nomes dos planetas
12) Podemos pausar o movimento dos planetas e satélites naturais, pressionando "p'
13) Com os botões do mouse, pode-se dar zoom in ou zoom out
14) Caso a iluminação esteja ativada, pode alternar em três tipos de iluminação, pressionando "I"
15) Caso a iluminação esteja desligada, pode ligar ou desligar a luz emitida pela estrela (Sol)
16) Pode-se mexer o "olhar da câmera", acionando algumas teclas
